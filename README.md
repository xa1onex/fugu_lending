# Fugu Lending

Modern landing page for Fugu Casino with high-end aesthetics, glassmorphism, and smooth animations.

## Features
- **Rich Aesthetics**: Premium dark theme with gold accent colors.
- **Glassmorphism**: Modern UI elements with backdrop filters.
- **Dynamic Ticker**: Real-time winners display.
- **Responsive Design**: Fully optimized for mobile and desktop.
- **Local Assets**: All key icons and images are served locally for better performance.

## Structure
- `/index.html` - Main landing page.
- `/css/styles.css` - Custom styles.
- `/js/script.js` - Interactive elements.
- `/assets/` - Project assets (icons, images).

## Development
To run the project locally, just open `index.html` in your browser.
